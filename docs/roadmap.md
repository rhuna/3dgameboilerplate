# FireWizard3D Development Roadmap

This roadmap outlines the planned evolution of the simulation engine.

---

# Phase 1 — Core Simulation (Completed)

✔ ECS architecture  
✔ deterministic simulation loop  
✔ environment simulation  
✔ movement system  
✔ replay recording  
✔ save/load snapshots  
✔ metrics system  

---

# Phase 2 — Simulation Depth

Planned additions:

- AI decision system
- resource gathering
- population growth
- simple economy
- faction relationships

Goal: emergent simulation behavior.

---

# Phase 3 — Gameplay Systems

Planned gameplay features:

- combat system
- abilities/spells
- inventory
- item drops
- NPC behaviors

---

# Phase 4 — World Simulation

Advanced simulation features:

- biome generation
- seasons and climate
- city formation
- trade networks
- migration

---

# Phase 5 — Visualization and Analysis

Improve observability.

Planned tools:

- replay scrubbing
- timeline viewer
- metrics graphs
- simulation inspector
- debugging overlays

---

# Long-Term Goals

FireWizard3D may evolve into:

- an emergent simulation sandbox
- a strategy game engine
- a research platform for AI and agent systems

---

# Architectural Principles

Future development should maintain:

- deterministic simulation
- ECS architecture
- reproducibility
- strong testing
- modular systems
